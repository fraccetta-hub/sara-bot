❓ Ayuda — Comandos del propietario

Enviá estos mensajes desde tu WhatsApp al número de Sara Bot para gestionar el negocio sin abrir el panel web.

📦 Gestión del catálogo

CATALOGO Ver todos los productos con stock y precio actuales
STOCK nombre_producto CANTIDAD Actualizar el stock de un producto. Ej: STOCK Rosas Rojas 25
PRECIO nombre_producto MONTO Cambiar precio en guaraníes. Ej: PRECIO Rosas Rojas 160000
AGOTADO nombre_producto Marcar un producto como agotado. Sara dejará de ofrecerlo
DISPONIBLE nombre_producto Volver a activar un producto agotado
NUEVO nombre | categoría | precio | stock | descripción Crear un producto nuevo. Ej: NUEVO Girasoles|Ramos|80000|10|Girasoles frescos

📸 Fotos de productos

NOMBRE nombre_producto Enviar esta frase + una foto en el mismo mensaje. La foto se guarda y Sara la mostrará a los clientes cuando pregunten por ese producto

💡 También podés subir fotos desde el panel web → Catálogo → editar producto.

🛒 Pedidos y control de chat

CONFIRMAR Confirmar el último pedido pendiente. Sara notifica al cliente
CANCELAR Cancelar el último pedido pendiente. Sara notifica al cliente
CHAT Tomar control del chat con el cliente. Todo lo que escribas llega directamente al cliente (Sara no interviene)
FIN Devolver el control a Sara. El bot retoma la conversación
¿Cómo funciona el takeover? Cuando llega un pedido, Sara te avisa por WhatsApp con los detalles. Respondé CONFIRMAR o CANCELAR para gestionar el pedido, o CHAT para hablar directamente con el cliente.

ℹ️ Otros comandos

AYUDA Ver todos los comandos disponibles directamente en WhatsApp

💡 Los comandos no distinguen mayúsculas/minúsculas. Podés escribir "catalogo", "CATALOGO" o "Catalogo".
