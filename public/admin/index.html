📌 Cómo usar:

  1. Exportá el CSV del catálogo para ver los nombres exactos de tus productos.
  2. Renombrá cada foto con el nombre del producto (ej: rosa-roja.jpg).
  3. Comprimí todas las fotos en un archivo ZIP.
  4. Subí el ZIP acá — las fotos se asignan automáticamente.

📋 Límites del ZIP:

  • Formatos aceptados: .jpg / .jpeg / .png / .webp
  • Máximo 300 imágenes por ZIP
  • Máximo 50 MB por archivo ZIP
  • Cada imagen: máx. 8 MB sin comprimir
